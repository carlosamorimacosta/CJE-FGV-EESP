# CJE-FGV-EESP
Este repositório reúne os principais projetos, análises e contribuições que desenvolvi durante minha trajetória na **Consultoria Júnior de Economia (CJE)** , a Empresa Júnior de Mercado Financeiro da **Fundação Getulio Vargas (FGV)** .

## 📁 Conteúdo

Aqui você encontrará uma coleção dos meus trabalhos, organizados para servir como portfólio e registro em nuvem:

- **Códigos em Python:** Scripts desenvolvidos para o processo de **Equity Research da WEG**, incluindo automação de coleta de dados e modelagem financeira.
- **Valuation:** Modelagens financeiras detalhadas (DCF, Múltiplos).
- **Relatórios e Apresentações:** Versões finais de análises e pitches realizados durante consultorias.
- **Resumos:** Sínteses de estudos de mercado e materiais de capacitação.
- **Análises Macroeconômicas:** Acompanhamento de indicadores e cenários econômicos.
